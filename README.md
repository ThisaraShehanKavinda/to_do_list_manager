# To-Do List Manager

A minimal and modern To-Do List Manager built with React.  
Easily add, edit, delete, and sort tasks with a clean and responsive UI.

## Features

- Add new tasks
- Edit existing tasks
- Delete tasks
- Sort tasks
- Minimal and visually appealing design
